enum class GameScreen

Represents the current UI/State of the application.

Dictates which "Screen" class is currently updating and rendering.

Enumerator
MainMenu 

The starting screen with options to start game or quit.

Playing 

The main active gameplay loop.

Paused 

The game loop is frozen, showing the pause menu.

LevelCompleted 

The screen shown between levels (Success state).

GameOver 

The screen shown when the player dies (Failure state).

Upgrades 

The shop/upgrade screen.

Quit 

Signal to the engine to close the application window.
