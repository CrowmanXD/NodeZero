Represents the lifecycle state of a game entity (Node).

Enumerator
Inactive 

The node is in the object pool but not currently active in the world.

Active 

The node is alive, updating, and rendering.

Dead 

The node has been destroyed (health <= 0) and is awaiting cleanup.
