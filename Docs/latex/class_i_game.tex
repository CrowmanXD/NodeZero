\doxysubsection{Detailed Description}
Central hub interface for the game application. 

Acts as a Mediator between different services (Score, Health, Spawning) and the main game loop. It also implements the \doxylink{class_subject}{Subject} interface to broadcast events to the UI and Audio systems.
