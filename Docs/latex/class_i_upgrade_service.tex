\doxysubsection{Detailed Description}
Interface for the game\textquotesingle{}s upgrade and progression system. 

Handles the logic for purchasing permanent player improvements using currency (points). It manages Health, Regeneration, Area of Effect (Zone), and Damage output upgrades.
