\doxysubsection{Detailed Description}
Static utility class for all game rendering operations. 

Encapsulates the logic for drawing geometric shapes, UI elements, and complex entity visuals (like the health-\/fill effect on nodes). Relies on Raylib for low-\/level drawing calls.
