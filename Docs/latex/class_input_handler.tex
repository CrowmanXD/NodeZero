\doxysubsection{Detailed Description}
Static wrapper around Raylib\textquotesingle{}s input system. 

Provides a unified interface for polling mouse and keyboard state. This abstraction allows for easier mocking during unit testing (if needed) or swapping the underlying input engine later.
