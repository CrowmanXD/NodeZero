\doxysubsection{Detailed Description}
Concrete implementation of the \doxylink{class_i_subject}{ISubject} interface. 

Maintains a thread-\/unsafe list of observers. Note\+: \doxylink{class_subject_a163cfae64cb06d87e74b36a499747356}{Notify()} is safe against observers detaching themselves during the update loop.
