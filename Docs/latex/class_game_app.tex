\doxysubsection{Detailed Description}
The Root Application Class. 

This class owns the OS Window context (Raylib Init\+Window), the main loop, and the top-\/level State Machine (Main \doxylink{class_menu}{Menu} -\/\texorpdfstring{$>$}{>} Playing -\/\texorpdfstring{$>$}{>} Pause). It acts as the container for all specific Screens and the Core \doxylink{class_game}{Game} instance.
