\doxysubsection{Detailed Description}
Handles the screen where the player can spend points to purchase upgrades. 

Draws current statistics and provides interactive buttons for purchasing persistent upgrades (Health, Regen, Damage, Zone Size). Handles mouse click debouncing and purchase logic.
