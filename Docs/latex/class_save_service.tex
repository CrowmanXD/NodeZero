\doxysubsection{Detailed Description}
Concrete implementation of the save system using a simple text-\/based format. 

This service handles file I/O, platform-\/specific path resolution (App\+Data on Windows, Home on Linux), and serialization of the \doxylink{struct_save_data}{Save\+Data} structure.
