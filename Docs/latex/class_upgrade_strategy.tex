\doxysubsection{Detailed Description}
Abstract base class for the Strategy Pattern used in player upgrades. 

Defines the contract for different types of upgrades (Health, Regen, etc.), allowing the \doxylink{class_upgrade_service}{Upgrade\+Service} to process purchases generically.
